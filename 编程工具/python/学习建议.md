1. **锁定 CPython**：忘掉其他解释器。
    
2. **短期目标 (Web 转型)**：学习 **FastAPI** (Python) + **Type Hints**。
    
    - PHP 8 也开始推强类型了，Python 3.10+ 的类型提示非常强大。
        
    - FastAPI 的性能极高（基于 Starlette），且大量使用异步（Async/Await），这是 PHP 程序员需要转弯的一个思维点（PHP 主要是同步阻塞模型）。
        
3. **长期目标 (AI 应用工程师)**：
    
    - **不要去学怎么从头写神经网络**（那是数学家的事）。
        
    - **学 LangChain / Semantic Kernel**：这是 Python 连接大模型（LLM）的中间件。
        
    - **核心逻辑**：利用 Python 极其丰富的生态，把业务数据（从 PHP/SQL 里的）清洗整理，喂给大模型，再把结果返回给用户。
        
    - **为什么是 Python？** 因为所有大模型的 API、SDK、向量数据库（Vector DB）的客户端，首发全是 Python（CPython）。
